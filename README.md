# Slate UI for Age of Conan

This repository contains two versions of **Slate UI**, a custom interface for *Age of Conan: Unchained*.

## Repository Contents

### Slate UI

The standalone version of Slate UI.

Use this version for a direct manual installation without the Strange UI installer framework.

The package includes several hotbar layouts and visual variants designed around different key configurations.

### Strange UI + Slate

Slate UI integrated into the Strange UI installer and supporting framework.

The included installer has been modified to support **Slate UI only**. All other UI selections from the original Strange UI installer have been disabled.

## Installation

Open the folder for the version you want to use and follow the instructions in its included README.

Back up any existing files under:

```text
Age of Conan\Data\Gui\Customized
```

before installing or replacing a customized interface.

## About Slate UI

Slate UI was originally designed as a visual hotkey-mapping interface, with layouts built around devices such as the Logitech G13 and expanded hotbar configurations.

Available packages may include:

* 59-key layouts
* 41-key layouts
* 22-key layouts
* Solid variants
* Transparent variants
* Updated HUD and bottom-bar components
* Pet Command Bar and Soul Fragment Bar fixes
