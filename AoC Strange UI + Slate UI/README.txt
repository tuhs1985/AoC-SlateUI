# Strange UI + Slate

A maintained combination of **Strange UI** and **Slate UI** for *Age of Conan: Unchained*.

This package uses the Strange UI installer and supporting files while installing the Slate interface and associated layout modifications.

## Installer Notice

The included installer currently supports **Slate UI only**.

All other UI selections from the original Strange UI installer have been disabled. Options shown within the installer apply only to available Slate layouts and variants.

## Features

* Slate-based interface
* Strange UI installer framework
* Multiple Slate hotbar layouts
* Solid and transparent variants
* Compatibility fixes for the current game client
* Updated bottom-bar positioning
* Fixed Soul Fragment and Pet Command Bar interaction
* Bundled installation and configuration files

## Installation

1. Download the latest release.
2. Extract the archive.
3. Run the included installer.
4. Select the desired Slate layout.
5. Confirm the *Age of Conan* installation directory.
6. Complete the installation and launch the game.

The installer places the customized interface files under:

```text
Age of Conan\Data\Gui\Customized
```

Back up any existing customized UI files before installation.

## Available Layouts

The installer contains Slate layout variants based on:

* 59 hotkeys
* 41 hotkeys
* 22 hotkeys
* Solid or transparent styling

Availability may vary by release.
